hexo使用

1.Hexo 安装

安装后,使用两种方式执行hexo

  1. npx hexo <command>
  2. 将Hexo所在的目录下的node_modules添加到环境变量中即可直接使用

echo 'PATH="$PATH:./node_modules/.bin"' >> ~/.profile

2.hexo写作

2.1新建文章

hexo new [layout] <title> # hexo new 'first'

布局(layout)

不同的布局对应不同的路径

布局 路径
post source/_posts //默认路径
page source
Draft source/_drafts

post –> source/_posts //默认路径

page –>source

Draft source/_drafts
